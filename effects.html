Portée et héritage en CSS

Héritage

Ce paragraphe a du texte en emphase à l’intérieur.

Code CSS :
.legacy { all: revert; border: medium solid; font-size: 200%; font-weight: bold; } .legacy em { border: inherit; }

Dans cet exemple de code, nous utilisons la propriété de raccourci ‘all’ qui va permettre d’appliquer sa valeur à toutes les propriétés. En utilisant cette approche, le style des paragraphes pour la propriété ‘font’ aura sa valeur par défaut. La taille est alors doublée, et la police d’écriture est alors mise en bold (gras). Le mot clé ‘inherit’ est appliqué à la propriété border. Ce mot clé impose un héritage. En effectuant ce scénario de code, nous obtenons une bordure supplémentaire autour du texte avec une référence d’emphase (mise en évidence visuelle).

Portée des effects CSS

code CSS :
p { color: #000091; font-size: 1.2rem; }

Dans ce code, nous appliquons un effet à tous les paragraphes d’une page. Le sélecteur utilisé est ‘p’, nous utilisons le sélecteur pour définir la portée de notre effet sur la page. En appliquant ce code, tous les paragraphes de notre page auront un style de couleur et une taille définis.


Code CSS :
button:hover { background-color: green; color: white; }

Ce code représente un exemple de pseudo-classe. Dans cet exemple, nous utilisons la pseudo-classe ‘:hover’. Cette pseudo-classe permet d’appliquer un effet lorsqu’un élément précis est survolé par la souris. Dans notre scénario, nous appliquons l’effet qui est appliqué à l’élément “button”.

Les effects avancés en CSS

Les effets avancés en CSS sont des techniques et des approches plus complexes permettant de styliser les éléments d’une page. Les fonctionnalités avancées de ces effets CSS nécessitent souvent une compréhension approfondie des bases de CSS. Les effets avancés en CSS sont souvent construits en utilisant plusieurs propriétés CSS combinés. L’association de plusieurs effets CSS permet également la création d’effet CSS avancés.

Les effets avancés CSS permettent donc de combiner avec précision plusieurs propriétés afin d’obtenir l’effet souhaité. Ils nous permettent d’ajouter des styles visuels élaborés ainsi que des mises en page et des interactions visuelles créatives poussées. Utiliser des effets CSS avancés offre également la possibilité de se démarquer en créant des contenus et des designs uniques et innovants.

element
parallax element
Code CSS : .parallax-container { perspective: 150px; } .parallax-element { transform: translateZ(-50px) scale(1.5); border: 2px solid black; border-radius: 50%; width: 5rem; height: 5rem; text-align: center; padding: .2em; }

Cet exemple de code illustre l’utilisation de l’effet de parallax. Le parallax crée une illusion de profondeur en faisant défiler différents éléments à différentes vitesses. Nous utilisons une classe ‘.parallax-container’ pour envelopper les éléments qui auront l’effet souhaité.
La propriété ‘perspective’ lui est associée, elle indique en valeur la distance avec laquelle l’utilisateur voit l’effet. Dans ce scénario, la valeur est définie à ‘150px’ ce qui signifie que le parallax sera visible à cette distance par l’utilisateur.
La seconde classe ‘.parallaxe-element’ est appliquée aux éléments qui vont subir l’effet. La propriété ‘transform’ qui lui est appliquée permet d’effectuer une transformation 3D à l’élément. La seconde propriété ‘ translateZ’ est utilisée pour déplacer l’élément sur l’axe Z (profondeur) dans l’espace.
L’ajout de valeur négative, tel que ‘-50px’ indique un déplacement vers l’arrière pour créer l’illusion de profondeur voulue.
La propriété ‘ scale’ termine ce code. Elle est utilisée afin d’augmenter la taille de l’élément, créant un effet supplémentaire, une impression d’accentuation.
