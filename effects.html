Portée et héritage en CSS

Héritage

Ce paragraphe a du texte en emphase à l’intérieur.

Code CSS :
.legacy { all: revert; border: medium solid; font-size: 200%; font-weight: bold; } .legacy em { border: inherit; }

Dans cet exemple de code, nous utilisons la propriété de raccourci ‘all’ qui va permettre d’appliquer sa valeur à toutes les propriétés. En utilisant cette approche, le style des paragraphes pour la propriété ‘font’ aura sa valeur par défaut. La taille est alors doublée, et la police d’écriture est alors mise en bold (gras). Le mot clé ‘inherit’ est appliqué à la propriété border. Ce mot clé impose un héritage. En effectuant ce scénario de code, nous obtenons une bordure supplémentaire autour du texte avec une référence d’emphase (mise en évidence visuelle).

Portée des effects CSS

code CSS :
p { color: blue; font-size: 16px; }

Dans ce code, nous appliquons un effet à tous les paragraphes d’une page. Le sélecteur utilisé est ‘p’, nous utilisons le sélecteur pour définir la portée de notre effet sur la page. En appliquant ce code, tous les paragraphes de notre page auront un style de couleur et une taille définis.


Code CSS :
button:hover { background-color: green; color: white; }

Ce code représente un exemple de pseudo-classe. Dans cet exemple, nous utilisons la pseudo-classe ‘:hover’. Cette pseudo-classe permet d’appliquer un effet lorsqu’un élément précis est survolé par la souris. Dans notre scénario, nous appliquons l’effet qui est appliqué à l’élément “button”.
